# Admin-Dashboard

A modern admin dashboard built as part of **The Odin Project's Intermediate HTML and CSS curriculum**. This project focuses on creating complex page layouts using CSS Grid, nested grid containers, and SVG icons while maintaining a clean and organized user interface.

## Overview

The dashboard recreates a typical administrative interface featuring navigation, project management cards, announcements, and trending users. The primary goal was to strengthen my understanding of CSS Grid and practice building a complete multi-section layout from scratch using only HTML and CSS.

## Features

* Sidebar navigation with SVG icons
* Search bar and notification section
* User profile and greeting area
* Custom CSS avatar system using initials
* Project cards with action icons
* Announcements panel
* Trending users section
* Modern card-based design
* Grid-based page layout
* Nested CSS Grid implementation throughout the application

## Technologies Used

* HTML
* CSS
* SVG Icons (Material Design Icons)

## Skills and Concepts Practiced

* CSS Grid layout design
* Nested grid containers
* Semantic HTML structure
* SVG integration and styling
* Component-based page organization
* Grid alignment and positioning
* Use of flexible sizing with `fr` units
* Custom avatar creation using CSS
* Visual hierarchy and spacing
* Card-based UI design

## What I Learned

Through this project I learned how to:

* Structure a complex dashboard layout using semantic HTML
* Use CSS Grid to build both page-level and component-level layouts
* Organize content using nested grids
* Integrate and style inline SVG icons
* Create reusable UI components with CSS
* Build a polished interface without relying on external frameworks
